# A-Star-Algorithm
 A*-Algorithm Visualisation in Python.
*Visualizes steps*

* random walls, start, goal
* diagonal movment

* create own map
   * *first start*
   * *then goal*
   * *then walls*
   * *remove by clicking again*
* click return to start search
    * *if start or goal is missing -> random map*

* Tkinter
